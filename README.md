# WP Webmaster Role

**Versión:** 1.0  
**Autor:** SEDICI 

## Descripción

Este plugin de WordPress crea un nuevo rol de usuario llamado **Webmaster** y permite activarlo o desactivarlo de forma manual en todo un multisitio. Este rol que hereda todas las capacidades del rol **Editor**, y además incorpora algunas capacidades avanzadas propias del rol **Administrador**.

Al desactivar el plugin, todos los usuarios que posean el rol de **Webmaster** son reasignados automáticamente al rol de **Editor**.

## Requisitos

- Wordpress 6.x o superior
- PHP 8.2 o superior
